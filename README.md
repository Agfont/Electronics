# Electronics
Electronics course during the 2019–2020 school year at the University of Barcelona.

## Content
- [Syllabus](https://www.ub.edu/pladocent/?cod_giga=364297&curs=2019&idioma=cat)

## Structure of the repository
- `practiques/`: Practical exercises and labs.
- `problemes/`: problems and exercises for practice.
- `examenes/`: exams and solutions.
- `teoria/`: Theoretical content of the course.

## Technologies
| Technology | Purpose |
|---|---|
| MSP432P401R LaunchPad Development Kit | Target microcontroller board used in labs |
| Energia IDE | Arduino-like development environment for TI LaunchPad |
| C | Primary programming language |
| Arduino libraries and tools | Libraries and utilities compatible with Energia |
